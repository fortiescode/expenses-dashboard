## Mission
Build a personal finance expense tracker dashboard using Next.js, hosted on Netlify.

## Tech Stack
- **Framework:** Next.js 15 (App Router, TypeScript)
- **Styling:** Tailwind CSS
- **Database:** SQLite (dev) / PostgreSQL (prod via Neon or Supabase)
- **Charts:** Recharts
- **Deployment:** Netlify
- **Auth:** NextAuth.js (optional Phase 2)

## Project Phases

### Phase 1 — Foundation
- PM writes specs
- Designer creates UI system
- Scaffold components

### Phase 2 — Core Build
- Frontend: Dashboard, expense forms, category views
- Backend: API routes, database schema
- Charts & visualizations

### Phase 3 — Ship
- DevOps: Netlify config, CI/CD
- QA: Testing, bug fixes
- Launch
